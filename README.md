# Usage
<!-- usage -->
```sh-session
$ npm install -g pushtag
$ pushtag COMMAND
running command...
$ pushtag (-v|--version|version)
pushtag/0.1.0 win32-x64 node-v12.16.3
$ pushtag --help [COMMAND]
USAGE
  $ pushtag COMMAND
...
```
